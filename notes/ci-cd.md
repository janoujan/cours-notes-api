# CI/CD

Le CI/CD (Continuous Integration / Continuous Deployment ou Delivery) est une méthode de développement logiciel qui automatise les étapes de préparation des applications.

- **CI (Intégration Continue)** : Pratique consistant à fusionner les modifications de code dans une branche partagée plusieurs fois par jour, déclenchant des tests automatisés pour détecter les erreurs rapidement.
- **CD (Distribution ou Déploiement Continu)** : Automatisation de la livraison du code vers des environnements de test ou de production après la phase de CI, permettant des mises à jour rapides et fiables.
